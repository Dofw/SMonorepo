## 库的选择

1. 命令行交互信息
  - inquirer 大生态用的比较多, 依赖比较重。
  - prompts 依赖比较少相对轻量。比较适合个人日常开发使用。(选择它)

2. 定义指令
  - commander node.js 命令行解决方案。

3. 辅助
  - chalk 字体样式
  - ora 字体样式
  - download-git-repo github下载仓库工具
  - handlebars 模板引擎
